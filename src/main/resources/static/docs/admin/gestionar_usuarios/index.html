Gestionar Usuarios

Crear, editar y eliminar perfiles de Administradores y Médicos con control de roles.

Introducción​

La sección Gestionar Usuarios permite al administrador del sistema crear, editar y eliminar usuarios dentro de la plataforma de telemedicina del proyecto IA - VPH.
Este módulo es fundamental para garantizar que solo personal autorizado tenga acceso a la aplicación y que sus roles estén correctamente definidos.

Dentro de esta sección se pueden administrar dos tipos principales de usuarios:

  • Administrador
  • Médico

Acceso a la Funcionalidad​

Desde la página principal del sistema, se puede acceder a Gestionar Usuarios mediante el botón correspondiente:

Vista de Usuarios Registrados​

Al ingresar a la sección, se muestra una lista con los usuarios ya registrados en el sistema, junto con su nombre, usuario y rol.
Cada registro permite editar o eliminar al usuario mediante los íconos de lápiz ✏️ y papelera 🗑️.

Crear un Administrador​

Para añadir un nuevo usuario administrador:

  1. Hacer clic en el botón rojo con el símbolo ➕ en la esquina inferior derecha.
  2. Seleccionar el rol Administrador.
  3. Completar los siguientes campos:
    • Nombre
    • Usuario
    • Contraseña
  4. Hacer clic en Guardar para registrar al administrador.

Crear un Médico​

El registro de médicos requiere información más detallada.

  1. Hacer clic en el botón rojo con el símbolo ➕ en la esquina inferior derecha.
  2. Seleccionar el rol Médico.
  3. Completar los siguientes campos:
    • Nombre
    • Usuario
    • Correo
    • Especialización
    • Sexo
    • N° de Registro
    • Contraseña
  4. Hacer clic en Guardar para registrar al médico en el sistema.

Funciones Disponibles​

  • Crear usuarios: Administradores y Médicos.
  • Editar usuarios: Modificación de los datos de un usuario existente.
  • Eliminar usuarios: Eliminación permanente de registros.
  • Gestión de roles: Definición clara de privilegios entre administradores y médicos.

Conclusión​

El módulo Gestionar Usuarios es esencial para el control de acceso y la administración del sistema de telemedicina.
Permite un manejo seguro y eficiente de los perfiles de Administradores y Médicos, asegurando el correcto funcionamiento de la plataforma en el marco del proyecto IA - VPH.
