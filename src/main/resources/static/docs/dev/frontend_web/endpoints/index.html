Frontend Web - Endpoints Principales

El proyecto IA - VPH utiliza un backend en Spring Boot que expone endpoints REST para usuarios administradores y médicos. El frontend web (Flutter) consume estos servicios a través de ApiService, AuthService y UbicacionService.

Leyenda de autenticación (Auth)​

En las tablas de endpoints se utiliza la columna Auth para indicar los requisitos de autenticación y permisos:

  • ✖️ → No requiere autenticación (público).
  • ✔️ → Requiere autenticación con token válido (JWT).
  • △ → Acceso restringido según el rol del usuario:
    • ADMIN → Acceso completo.
    • DOCTOR → Acceso limitado a funciones médicas.
