Preguntas Frecuentes

1) ¿Necesito Internet para usar SISA?​

Sí. Para ver el video, usar el chat, registrar el automuestreo, recibir notificaciones y cargar datos, necesitas conexión a Internet.

2) ¿La app funciona en iOS?​

Por ahora solo Android 8+ mediante APK.

3) ¿La app guarda mis datos personales?​

Sí, pero de forma segura y confidencial. La información se usa exclusivamente para tu proceso de atención y mejora del servicio.

4) ¿Cómo registro el dispositivo de automuestreo?​

En Automuestreo → Registrar dispositivo. Sigue los pasos de la pantalla y verifica que la información se guarde correctamente.

5) ¿Dónde entrego mi muestra?​

En los GAD o centros de salud autorizados.

6) No me llegan notificaciones, ¿qué hago?​

Verifica en Android:

  • Ajustes del sistema → Notificaciones → SISA (permitidas).
  • Conexión a Internet y ahorro de batería desactivado para SISA.

7) ¿Cómo recuperar mi contraseña si la olvidé?​

En la pantalla de Iniciar Sesión, usa la opción de Olvidé mi contraseña.

8) ¿Puedo borrar mi cuenta?​

Sí. Solicítalo al equipo del proyecto para proceder con la eliminación de datos conforme a las políticas vigentes.

Si tienes otra duda, no dudes en comunicarte con el equipo del proyecto IA - VPH.
