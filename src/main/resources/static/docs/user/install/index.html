Instalación de la App

Requisitos previos​

  • Dispositivo Android 8.0 o superior
  • Conexión a Internet para descargar el APK y usar las funciones en línea

Pasos de instalación (APK)​

  1. Descarga el archivo APK provisto por el equipo del proyecto.
  2. En tu teléfono, habilita:
    Ajustes → Seguridad → Instalar apps de orígenes desconocidos (o “Permitir desde esta fuente”).
  3. Abre el APK y toca Instalar.
  4. Busca SISA en tu menú de apps y ábrela.

Primer inicio​

  • Si ya tienes cuenta: Iniciar Sesión.
  • Si es tu primera vez: Crear una cuenta y completa los datos solicitados.
