Manual de Usuario

A continuación, un recorrido por las secciones principales de SISA.

1) Automuestreo​

Este módulo te guía para realizar el automuestreo vaginal para VPH.

Qué encontrarás

  • Video tutorial con instrucciones claras.
  • Registrar dispositivo de automuestreo.
  • Pasos guiados para realizar la prueba.

Importante: Entregar la muestra en el centro de salud o GAD indicado para su procesamiento.

2) Contenidos educativos​

Accede a materiales validados sobre:

  • Prevención de embarazo y métodos anticonceptivos
  • Salud sexual integral y autocuidado
  • VIH: prevención, diagnóstico y tratamiento

3) Localización de servicios​

Ubica en el mapa los puntos de atención cercanos.
Filtros disponibles:

  • Ginecología
  • En caso de agresión
  • Psicología

4) Notificaciones y recordatorios​

Recibe avisos para:

  • Realizar el automuestreo.
  • Entregar la muestra en el punto asignado.
  • Confirmar si ya completaste el proceso.

5) Asistente virtual (SISA Chat)​

Consulta al chat sobre:

  • Prevención del VPH.
  • Cáncer de cuello uterino (CCU).
  • Autocuidado en salud sexual.
  • Mucha cosas más.

Buenas prácticas​

  • Lee y sigue el video antes de iniciar el automuestreo.
  • Conserva la muestra según las instrucciones del kit.
  • Usa el mapa para ubicar puntos de localización de servicios relacionados.
  • Mantén activas las notificaciones para no olvidar pasos clave.
